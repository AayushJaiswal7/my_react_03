# State-App

A basic React application bootstrapped with [Vite](https://vitejs.dev/). This project demonstrates basic state management and component rendering in React, featuring a `LudoBoard` component.

## Prerequisites

Before running this project, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16 or higher is recommended)
- npm (which comes with Node.js)

## Getting Started

1.  **Navigate to the project directory:**
    ```bash
    cd basic-state-app2
    ```

2.  **Install the dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the application in your browser:**
    By default, the Vite development server runs on `http://localhost:5173/`. Check your terminal output for the exact URL.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.
Open [http://localhost:5173/](http://localhost:5173/) to view it in your browser.
The page will reload when you make changes.

### `npm run build`

Builds the app for production to the `dist` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`

Locally previews the production build generated by `npm run build`.

### `npm run lint`

Runs ESLint to analyze the code for errors and formatting issues.

## Project Structure

- `src/`: Contains the main source code of the application.
  - `App.jsx`: The main application component.
  - `LudoBoard.jsx`: A sample component demonstrating state management.
  - `main.jsx`: The entry point for the React application.
- `public/`: Contains static assets like favicons.
- `vite.config.js`: Configuration file for Vite.
